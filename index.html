Oct 2025 Deep Learning

Early Wheat Disease Detection — Deep Learning Classifier

A transfer learning classifier on ResNet-18 (ImageNet pre-trained) in PyTorch; froze convolutional layers and replaced the FC head for disease-specific classification on 224×224 RGB inputs, cutting training time versus scratch initialization.

  • Built a preprocessing and augmentation pipeline (PIL, OpenCV) on the Mendeley Wheat Disease dataset — random flips, rotations, color jitter — to improve generalization and address class imbalance
  • Evaluated with accuracy, precision, recall, F1-score, and confusion matrix via Scikit-Learn; TorchVision CNN weights provided strong feature extraction baseline for early-stage disease patterns
