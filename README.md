# Portfolio

A clean, responsive personal portfolio website for Gaurav Sharma built with plain HTML, CSS, and JavaScript.

## Overview

This repository contains a static landing page showcasing:
- Hero introduction and contact call-to-action
- About section with skills and domain highlights
- Project showcase with feature summaries and GitHub links
- Education and experience timeline
- Contact section with email, phone, and GitHub links

## Features

- Responsive desktop and mobile navigation
- Hamburger menu for mobile view
- Smooth scrolling to section anchors
- Active navigation highlight based on scroll position
- Scroll reveal animations for page sections
- Footer year auto-updates to the current year

## Tech stack

- HTML5
- CSS3
- JavaScript (vanilla)


## Notes

- This is a static portfolio site with no backend or build step.
- All content and styling are contained in the repository files.
- For updates, edit the HTML content and CSS/JS files directly.
